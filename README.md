# Personal Resume Website

A personal resume website showcasing my education, skills, projects, achievements, and contact information.

## Features

- Responsive design
- Professional resume layout
- About Me section
- Education section
- Skills section
- Projects section
- Achievements section
- Contact section
- Smooth navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
